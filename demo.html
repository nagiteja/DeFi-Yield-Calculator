Technical Implementation

APR to APY Conversion:

APY = (1 + APR/365)^365 - 1
Example: 25% APR → 28.39% APY

Impermanent Loss Formula:

IL = 2√(price_ratio)/(1 + price_ratio) - 1
Where price_ratio = new_price / initial_price

Compounding Simulation:

Daily reinvestment of yields
Final value = Initial × (1 + daily_rate)^days

About This Tool

DeFi Yield Calculator is a lightweight, single-page web application built with pure HTML, CSS, and JavaScript. It provides comprehensive yield calculations for DeFi strategies and helps users understand impermanent loss risks through interactive simulation.

No build tools, no dependencies, no installation required. Just open the HTML file in any modern browser and start calculating!
